# mcp-client-gemini

To install dependencies:

```bash
bun install
```

To run the client:

```bash
bun run client.ts
```

To run the client and the server:


```bash
bun run client.ts server.ts
```

This project was created using `bun init` in bun v1.2.15. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
